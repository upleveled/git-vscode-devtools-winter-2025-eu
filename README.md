# VS code commands

delete line => `cmd/ctrl + shift + K`

copy line => `cmd/ctrl + C`

paste line => `cmd/ctrl + V`

commenting lines(German keyboard) `cmd + /` or `ctrl + /` or `ctrl + #`

multiple cursors `cmd/ctrl + D, option + click`
